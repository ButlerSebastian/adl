# Test Files

This directory contains temporary test files and validation scripts used during development.

## Contents

### Validation Scripts
- `check-circular.js` - Check for circular dependencies
- `check-schema.js` - Validate JSON schemas
- `check_circular_deps.js` - Alternative circular dependency checker
- `final_validation.py` - Final validation script
- `simple_validate.js` - Simple validation utility
- `test_validation.js` - Test validation script
- `validate-all.js` - Validate all examples
- `validate-simple.js` - Simple validation
- `validate_all_examples.js` - Validate all examples (alternative)
- `validate_examples.js` - Validate examples

### Test Artifacts
- `test-validation.adl` - Test ADL file
- `test-validation.d.ts` - Generated TypeScript definitions
- `test-validation.json` - Test JSON output
- `test.d.ts` - TypeScript definitions
- `test.json` - JSON test data

### Test Scripts
- `test_ref.py` - Reference test script
- `test_schema.py` - Schema test script
- `test_transformer_all.py` - Transformer test suite
- `test_transformer_issue.py` - Transformer issue test

## Note

These files are for development and testing purposes only. They are not part of the core ADL distribution.
